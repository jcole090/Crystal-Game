Crystal Colector

Instructions

You will be given a random number at the begining of the game. The objective of the game is to match that number using the crystals provided. You will do this by clicking on each crystal. The crystals will have random values that change each time the game restarts. each time you click on a crystal the value of that crystal will be added to you. Kind of like Blackjack. See if you can match the value.
